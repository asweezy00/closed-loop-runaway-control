# ML Seizure-Like Event Detection

This directory is a focused machine-learning extension for the BME 525 runaway
control project. It trains a supervised classifier to detect seizure-like
runaway windows from the simulated LFP proxy generated by the model-based
project.

The model is **not a clinical seizure detector**. It is a controlled simulation
study for evaluating whether ML features can separate runaway-like windows from
baseline, weak-runaway, clean no-runaway, noisy, and artifact-corrupted records.

## Approach

- Generate simulated LFP records from the existing E/I spiking network.
- Slice each record into overlapping windows.
- Label windows as positive when they overlap the induced runaway interval.
- Extract interpretable time-domain and spectral features.
- Compare several classifiers:
  - logistic regression
  - random forest
  - extra trees
  - histogram gradient boosting
- Select the model with the best validation ROC-AUC.
- Report holdout ROC-AUC, PR-AUC, sensitivity, specificity, precision, F1,
  confusion matrix, and the selected decision threshold.

## Run

From the `final_project` directory:

```bash
../.venv/bin/python ml_seizure_detection/scripts/run_ml.py --quick --out ml_seizure_detection/results/ml_run_quick
../.venv/bin/python ml_seizure_detection/scripts/run_ml.py --out ml_seizure_detection/results/ml_run --seeds 42-51
```
